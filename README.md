Bash script to automate the **GfxTablet** server connection.
Features:
- Auto-detect Local IP.
- System Notification.
- OneClick Open to draw quick.

How to Use:

Download the original application file named 'networktablet' from here (https://github.com/rfc2822/GfxTablet/releases)

Place the files in the same folder.

<img width="473" height="414" alt="image" src="https://github.com/user-attachments/assets/2190d546-ede1-4512-b0c3-b4c736787c67" />

You can now run the file and it will work. (Make sure you grant the necessary permissions.)

For beginners:
How to make it a launcher (send to desktop in Linux)

Right-click on the desktop

<img width="230" height="363" alt="Screenshot_2026-04-09_13-16-54" src="https://github.com/user-attachments/assets/6021e8fe-e851-49ac-9f88-3d1bea347f22" />

Write the name you want.

Write the comment you want (you can leave it blank)

Click the button as shown in the image to browse the files.

<img width="370" height="416" alt="Screenshot_2026-04-09_13-23-44 (1)" src="https://github.com/user-attachments/assets/cb41af4a-2fba-40be-a475-c35e94a4849c" />

Select the executable file 'gfxtablet.sh'.

<img width="630" height="626" alt="Screenshot_2026-04-09_13-26-05 (1)" src="https://github.com/user-attachments/assets/b6d5d772-eebf-41c2-9027-e28f286c5c42" />

Click Open.

Choose the icon you want (optional but recommended)
